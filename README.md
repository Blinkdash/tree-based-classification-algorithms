Проведено исследование древовидных алгоритмов классификации на медицинском датасете осложнений инфаркта миокарда. Выполнена предобработка данных, обучение модели Decision Tree, анализ влияния параметров max_depth и min_samples_leaf, подбор оптимальных гиперпараметров и оценка качества по accuracy и confusion matrix.
